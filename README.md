# PortScanner
A Python-based TCP port scanner that checks network services, identifies open/closed ports, and demonstrates socket programming and network troubleshooting concepts.
The project uses Python socket programming to establish TCP connections, detect open ports, and provide information about common network services such as SSH, HTTP, HTTPS, DNS, and FTP.

## Features

- Scan multiple TCP ports automatically
- Detect open and closed ports
- Identify common services by port number
- Uses socket programming
- Implements connection timeouts
- Provides a simple command-line interface

## Technologies Used

- Python
- TCP/IP Networking
- Socket Programming
- Command-Line Interface

## Networking Concepts Practiced

- IP addresses
- TCP connections
- Ports and services
- Network troubleshooting
- Client-server communication
